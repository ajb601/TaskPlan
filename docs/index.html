About Us

TaskPlan is a simple and intuitive app designed to help you manage your tasks effectively. Whether it's work-related projects or personal to-dos, TaskPlan has got you covered.

Features

  • Easy to Use Task Input
  • Reactive Travel Time Calculator
  • Creative Calendar To Keep You In Check!

How To Use

1. Download and Install

  • Download TaskPlan from the Play Store here.
  • Install the app on your device.

2. Create an Account

  • Open TaskPlan and create an account using your email and a secure password.

3. Add a Task

  • Tap the "Set Task" button to add a new task.
  • Fill in the task details, such as title, description, and due date.

4. Task List

  • View and manage your tasks on the main Task List screen.

5. Additional Features

  • Utilize the reactive travel time calculator for effective planning.
  • Add unlimited tasks to stay organized.

6. Tips

  • Tap on a task to edit or update its details.
  • Delete tasks when finished.
